零成本部署自己的ChatGPT
2023-08-27 11:05:28 #ChatGPT

注意事项

  • 需要一个Chatgpt的API
  • GitHub账号
  • Vercel账号（可以通过GitHub来注册）

1.寻找合适自己的项目

本文章采用ChatGPT-Next-Web来部署

有多个项目来选择chatgpt-web，chatgpt-vercel，ChatGPT-Next-Web，这里推荐ChatGPT-Next-Web

该项目交互性强，功能多

  • 登录GitHub，fork项目

  • 点击只复制主分支

    这样你就将该项目复制在自己的仓库了

    2.开始部署

  • 注册Vercel，用GitHub账号来绑定Vercel，成功后选择Add new project，并将项目上传.

  • 在项目配置页，点开 Environmane Variables 开始配置环境变量，并依次新增名为 OPENAI_API_KEY 和 CODE 的环境变量.

    • OPENAI_API_KEY中添加的是你自己账号的API，CODE添加是密码，当一切妥当后，便可以点击Deploy了，网站也会成功部署.

3.添加域名

  • 首先要买个属于自己的域名，我选择GoDaddy，因为国内域名注册及其麻烦，所以还不如用外国平台来购买注册.
  • 分别进入部署的项目->Settings->Domains然后会生成要求解析的类型、名称和值，到自己对应的域名控制台添加相应的解析记录

    当配置显示为这样，域名添加也就成功了
